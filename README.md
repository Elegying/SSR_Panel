# SSR_Panel

SSR_Panel is a monorepo that brings together the SSR and AnyTLS panel tooling:

- `ssr-admin-panel`: SSR user traffic and account management panel.
- `anytls-panel`: AnyTLS and multi-protocol node management panel.
- `ssr-server-optimizer`: one-command optimizer for legacy Python SSR servers.

Each subproject keeps its own README and implementation details. Work inside the relevant subdirectory when developing or deploying a specific component.
